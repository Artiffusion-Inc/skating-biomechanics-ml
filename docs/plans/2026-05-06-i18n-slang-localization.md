# i18n Localization with Slang Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use subagent-driven-development (recommended) or executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace all hardcoded Russian strings with type-safe generated translations via `slang`. Enable English fallback and allow AI agents to translate via JSON without touching Dart code.

**Architecture:** `slang` generates `Strings` class from `i18n/ru.i18n.json`. All UI widgets use `t.key.path` instead of literal strings. `MaterialApp` sets `locale` and `supportedLocales`. CI runs `dart run slang` before `flutter analyze`.

**Tech Stack:** `slang`, `slang_flutter`, `ast-grep` rules for hardcoded string detection

---

## File Structure

| File | Responsibility |
|------|----------------|
| `mobile/i18n/ru.i18n.json` | Source of truth for Russian translations |
| `mobile/i18n/en.i18n.json` | English fallback translations |
| `mobile/lib/i18n/strings.g.dart` | Generated by slang — **do not edit** |
| `mobile/lib/i18n/slang_util.dart` | Helper: `resolveLocale()`, `StringExtensions` |
| `mobile/lib/main.dart` | `MaterialApp` with `locale` and `localizationsDelegates` |
| `mobile/ast-grep/hardcoded-string.yml` | ast-grep rule to catch untranslated strings |
| `mobile/test/i18n/strings_test.dart` | Verify all keys load, no missing English fallback |

```dart
import 'package:flutter/material.dart';
import 'package:flutter_localizations/flutter_localizations.dart';

import 'providers/app_providers.dart';
import 'theme/app_theme.dart';
import 'routing/app_router.dart';

void main() {
  WidgetsFlutterBinding.ensureInitialized();
  LocaleSettings.useDeviceLocale();
  runApp(const AppProviders(child: EdgeSenseApp()));
}

class EdgeSenseApp extends StatelessWidget {
  const EdgeSenseApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp.router(
      title: 'EdgeSense Capture',
      theme: AppTheme.dark,
      routerConfig: buildRouter(),
      locale: TranslationProvider.of(context).locale,
      supportedLocales: LocaleSettings.supportedLocales,
      localizationsDelegates: GlobalMaterialLocalizations.delegates,
    );
  }
}
```

## Task 6: ast-grep Rule for Hardcoded Strings

**Files:**
- Create: `mobile/ast-grep/hardcoded-string.yml`

- [ ] **Step 1: Write ast-grep rule**

```yaml
id: hardcoded-string
language: dart
severity: warning
message: "Hardcoded string detected — use slang translation instead"
pattern:
  kind: string_literal
  regex: "[А-Яа-яЁё]+"
  inside:
    kind: argument_list
    inside:
      any:
        - kind: invocation_expression
        - kind: instance_creation_expression
      has:
        kind: type_identifier
        regex: "Text|AppBar|SnackBar|AlertDialog"
```

- [ ] **Step 2: Commit**

```bash
git add mobile/ast-grep/hardcoded-string.yml
git commit -m "chore(i18n): add ast-grep rule to detect untranslated Russian strings"
```

```dart
  group('Translations', () {
    test('all Russian keys have English fallback', () {
      final ru = Translations.ru;
      final en = Translations.en;

      // Ensure base locale loads
      expect(ru.ble.scanTitle, 'Подключение IMU');
      expect(en.ble.scanTitle, isNotNull);

    test('no empty translation values', () {
      final ru = Translations.ru;
      expect(ru.permissions.grant, isNotEmpty);
      expect(ru.ble.scan, isNotEmpty);
      expect(ru.ble.next, isNotEmpty);
    });
  });
}
```
