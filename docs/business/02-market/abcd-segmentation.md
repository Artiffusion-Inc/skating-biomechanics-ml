# ABCD-сегментация SkateLab

> Методология: ABCDX framework. A = платящие, B = с возражениями, C = низкая ценность, D = не покупают, X = хотим, но нет решения.

## Сегмент A — Платящие клиенты

### A1: Тренеры сборных и академий

| Поле | Значение |
|------|----------|
| Сегмент | Тренеры региональных/национальных сборных и топ-академий |
| Когда? | Подготовка к соревнованиям, разбор элементов после тренировки |
| Боль/задача | Нет объективных данных для корректировки техники. "Кажется, внешнее ребро" — но точно не знают. Споры с учениками. |
| Ущерб в деньгах | 15–35K ₽/мес на фигуриста (потерянное время, неэффективные тренировки, травмы). Тренировка 3500 ₽/час, 1/3 времени — разбор ошибок. |
| Ущерб (кроме денег) | Травмы из-за неправильной техники. Конфликты тренер-ученик. Отсутствие прогресса → ученик уходит. |
| Дедлайн? | Да — перед соревнованиями (сезонный) |
| Решение | SkateLab: видео → метрики → рекомендации на русском |
| Процесс решения | Загрузить видео → получить анализ за ~12с → показать ученику метрики и рекомендации |
| Ценность/выгода | Объективные данные. Сокращение времени разбора ошибок с 10–60 мин до мгновенного. Сравнение с эталоном. Трекинг прогресса. |
| Оффер | "Объективные метрики техники на русском. От видео до рекомендаций — за 12 секунд." |
| Цена | 2,500–5,000 ₽/мес (Coach Dashboard) |
| Контроль ценности | Да — метрики до/после, прогресс во времени |
| Кейс | Да — после интеграции в пилотных школах |

### A2: Продвинутые фигуристы-любители

| Поле | Значение |
|------|----------|
| Сегмент | Фигуристы уровня CSEP и выше, тренирующиеся 4+ раз/неделю |
| Когда? | После тренировки: "Правильно ли я сделал прыжок? Что исправить?" |
| Боль/задача | Тренер не всегда видит детали. Самоанализ невозможен без видео. Kinovea — ручной, долгий. |
| Ущерб в деньгах | Дополнительные тренировки для исправления ошибок: 5–15K ₽/мес |
| Ущерб (кроме денег) | Фрустрация от отсутствия прогресса. Риск травмы. Неопределённость. |
| Дедлайн? | Нет (постоянная потребность) |
| Решение | SkateLab Pro: загрузил видео → получил метрики + рекомендации |
| Процесс решения | Записал элемент на телефон → загрузил → получил анализ |
| Ценность/выгода | Сокращение пути к чистому элементу. Объективная оценка качества (GOE proxy 0–10). Сравнение с эталоном. |
| Оффер | "Узнай, что не так с твоим прыжком. Загрузи видео — получи анализ на русском." |
| Цена | 1,000–2,500 ₽/мес |
| Контроль ценности | Да — GOE proxy score, трекинг прогресса |
| Кейс | Да |

## Сегмент B — С возражениями

### B1: Хореографы-постановщики

| Поле | Значение |
|------|----------|
| Сегмент | Хореографы, ставящие программы для фигуристов |
| Когда? | Подготовка программы к сезону, подбор элементов под музыку |
| Боль/задача | Ручной подбор элементов. Нет инструмента для визуализации программы на катке. Нет анализа музыкальных сегментов. |
| Ущерб в деньгах | Часы ручной работы: 10–20K ₽ стоимость времени на программу |
| Ущерб (кроме денег) | Неоптимальные программы. Превышение лимита времени. |
| Дедлайн? | Да — начало сезона |
| Решение | Choreography Planner: ISU elements + CSP solver + music analysis |
| Процесс решения | Загрузить музыку → получить анализ (BPM, ключ, сегменты) → автогенерация программы → редактирование в DAW timeline |
| Ценность/выгода | Визуальное планирование. Автоматический подсчёт TES/TSS/PCS. SVG рендер катка. |
| Оффер | "Спроектируй программу под музыку. AI подберёт элементы, ты — финализируй." |
| Цена | 2,000–4,000 ₽/мес |
| Контроль ценности | Частично — TES/TSS/PCS считаются автоматически |
| Кейс | Нет (пока) |

**Возражения:** "Я уже ставлю программы вручную, зачем мне AI?" → Ответ: экономия 5–10 часов на программу, автоматический подсчёт баллов.

### B2: Родители юных фигуристов

| Поле | Значение |
|------|----------|
| Сегмент | Родители детей 6–14 лет, занимающихся фигурным катанием |
| Когда? | После тренировки ребёнка: "Как он прыгает? Прогресс есть?" |
| Боль/задача | Не понимают технику. Тренер говорит "лучше", но что это значит в цифрах? Тратят 35K+ ₽/мес на тренировки, не видят объективной отдачи. |
| Ущерб в деньгах | 35K+ ₽/мес на тренировки без понимания эффективности |
| Ущерб (кроме денег) | Тревога за ребёнка. Не могут помочь. Не понимают, что происходит. |
| Дедлайн? | Нет |
| Решение | SkateLab Family: прогресс-дашборд, рекомендации, сравнение с эталоном |
| Процесс решения | Записать ребёнка на телефон → загрузить → получить метрики → показать тренеру |
| Ценность/выгода | Понимание прогресса в цифрах. Объективные данные для диалога с тренером. Мотивация ребёнка (видит прогресс). |
| Оффер | "Видишь прогресс ребёнка в цифрах. Загрузи видео — получи анализ." |
| Цена | 500–1,500 ₽/мес |
| Контроль ценности | Да — трекинг прогресса, метрики |
| Кейс | Нет (пока) |

**Возражения:** "Зачем мне это, если тренер и так тренерует?" → Ответ: объективные данные, которые тренер не даёт. Защита инвестиций в обучение.

## Сегмент C — Низкая ценность

### C1: Начинающие фигуристы

| Поле | Значение |
|------|----------|
| Сегмент | Новички, первые шаги на льду, базовые элементы |
| Когда? | Редко — техника ещё не на уровне, где анализ имеет смысл |
| Боль/задача | Общие вопросы: "Правильно ли я катаюсь?" — слишком общий для ML-анализа |
| Ущерб в деньгах | Низкий — начинающие платят меньше за тренировки |
| Ущерб (кроме денег) | Разочарование, бросают спорт |
| Дедлайн? | Нет |
| Решение | Freemium: базовый анализ (скелетон, углы) |
| Процесс решения | Загрузить видео → получить базовую визуализацию |
| Ценность/выгода | Мотивация продолжать. Визуальный фидбек. |
| Оффер | Freemium: "Увидь свой скелетон на видео. Бесплатно." |
| Цена | 0 ₽ (freemium) → 300–500 ₽/мес |
| Контроль ценности | Низкий |
| Кейс | Нет |

**Проблема:** Много ресурсов на онбординг, мало платят. Техника ещё не на уровне для ML-рекомендаций.
