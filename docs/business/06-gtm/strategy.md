# Go-to-Market Strategy

## Positioning

**SkateLab** — AI-тренер по фигурному катанию. Объективные метрики и рекомендации на русском языке. От видео до анализа — за 12 секунд.

**Tagline:** "Твой прыжок в цифрах. Анализ видео за 12 секунд."

**Категория:** Sports Tech / AI Coaching / Biomechanics Analysis

## GTM Phases

### Phase 1: CustDev + Pilot (Q2–Q3 2026)

**Цель:** Валидация ценностного предложения, 3–5 пилотных школ, 50 beta-пользователей.

| Действие | Детали |
|----------|--------|
| CustDev интервью | 30+ интервью с тренерами, фигуристами, родителями |
| Пилотные школы | 3–5 школ в СПб (через контакты Алисы) |
| Бесплатный пилот | 1 мес бесплатного использования для пилотных школ |
| Сбор метрик | Metrik: время анализа, точность, NPS, retention |
| Итерация | Улучшение продукта по фидбеку |

**Каналы:** Telegram-сообщества тренеров, DUSSH/ФМШ контакты, конференции (Спорт-IT).

### Phase 2: Early Adopters (Q3–Q4 2026)

**Цель:** 100–200 платящих, Product-Market Fit подтверждение.

| Действие | Детали |
|----------|--------|
| Product Hunt / Habr | Launch статья на Habr ("AI-тренер по фигурному катанию") |
| Telegram канал | Контент: сравнение видео до/после, метрики, кейсы |
| Инфлюенсеры | Тренеры с подпиской → referral программа |
| SEO | "анализ фигурного катания видео", "биомеханика прыжка" |
| YouTube | Демо-видео: как загрузить видео → получить анализ |

**Реферальная программа:** Приведи тренера → 1 мес бесплатного Coach Dashboard.

### Phase 3: Scale (2027)

**Цель:** 500+ платящих, B2B partnerships.

| Действие | Детали |
|----------|--------|
| Федеральные partnership | ФФКР (Федерация фигурного катания России) |
| Аппаратная интеграция | EdgeSense IMU → SkateLab |
| Мобильное приложение | Flutter (частично готово — `mobile/lib/`) |
| Корпоративный блог | Кейсы, исследования, ML behind the scenes |
| Конференции | ISU Congress, Спорт-IT, AI в спорте |

## Messaging by Segment

### Тренеры (A1)
> "Объективные данные о техке ученика. Загрузи видео — получи метрики и рекомендации на русском. За 12 секунд."

### Фигуристы (A2)
> "Узнай, что не так с твоим прыжком. GOE-оценка, сравнение с эталоном, трекинг прогресса."

### Хореографы (B1)
> "AI подберёт элементы под музыку. Спроектируй программу на визуальном таймлайне."

### Родители (B2)
> "Видишь прогресс ребёнка в цифрах. Объективные данные для диалога с тренером."

## Pricing Strategy

1. **Freemium** — 3 бесплатных анализа/мес. Хватит, чтобы увидеть ценность.
2. **Monthly** — Для начинающих. Без lock-in.
3. **Annual (20% скидка)** — Для продвинутых. Снижает churn летом.
4. **B2B contract** — Для клубов и школ. Volume discount.

## Risks

| Риск | Mitigation |
|------|-----------|
| Тренеры не хотят tech | Пилот → показываешь ценность → они убеждаются |
| Сезонность (лето) | Annual plans, хореография (off-ice), контент |
| Холодные продажи B2B | CustDev-first → warm intros через Алису |
| Конкуренция от Omega | Omega = федерации, мы = фигуристы/тренеры. Разные сегменты. |
